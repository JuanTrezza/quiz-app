# QuizMaster

Un quiz interactivo de trivia hecho con **HTML, CSS y JavaScript puro**. Elegí una categoría, ajustá la dificultad y respondé preguntas con tiempo limitado para ver tu puntaje final.

## Vista rápida

QuizMaster está pensado para jugar rápido y sin complicaciones:

- Elegís una categoría temática.
- Seleccionás el nivel de dificultad.
- Respondés 10 preguntas con cuenta regresiva.
- Al terminar, ves tu resultado y un resumen de respuestas.

## Características

- Más de 6 categorías disponibles.
- Tres niveles de dificultad: fácil, medio y difícil.
- Temporizador por pregunta.
- Barra de progreso del avance.
- Pantalla final con puntaje y resumen.
- Diseño responsive y visualmente moderno.

## Categorías

- HTML5

## Cómo usarlo

1. Abrí el archivo `index.html` en tu navegador.
2. Elegí una categoría.
3. Seleccioná una dificultad.
4. Hacé clic en **¡Jugar!**.
5. Respondé las 10 preguntas antes de que se acabe el tiempo.

## Estructura del proyecto

- `index.html`: estructura de la interfaz.
- `style.css`: estilos y diseño visual.
- `app.js`: lógica del juego y banco de preguntas.

## Idea del proyecto

Este proyecto busca ofrecer una experiencia simple pero entretenida para practicar conocimientos generales con una interfaz clara, animaciones suaves y un flujo de juego fácil de seguir.

## Captura mental de la experiencia

Primero ves una pantalla de inicio con categorías y dificultad. Después entra el juego con progreso y temporizador. Al final aparece un resumen con tu puntaje y el detalle de las respuestas.

## Posibles mejoras
